# Tabs

Tabs component with different props.

## Example

<Canvas of={Tabs.Default} />

## Usage

```ts
import {Tabs} from "dragons-treasure";

const Example = () => {
  return (
    <Tabs
      tabs={[
        { title: "Tab 1", content: "Content 1" },
        { title: "Tab 2", content: "Content 2" },
        { title: "Tab 3", content: "Content 3" },
      ]}
    />
  );
};

export default Example;
```

## Arguments
- **tabs** - An array of tab content
